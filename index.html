Node.js Express 框架使用

关于 Express 的介绍和用法在Node.js Express 框架已经有了非常详尽的阐述。以下是在实际应用中需要注意的几点：

1. 静态文件

Express 提供了内置的中间件 express.static 来设置静态文件如：图片，CSS, JavaScript 等。但是， Express 默认使用 jade 模板，如果使用 html 模板，需要一些简单的配置。

  1. 安装 ejs

  2. 在 app.js 文件中引入 ejs

  3. 设置 html 引擎

  4. 设置视图引擎

  5. 设置静态文件路径，XXX 即为存放静态文件的目录

2. body-parser

body-parser 是 node.js 的一个中间件，用于处理 JSON, Raw, Text 和 URL 编码的数据。

  • 如果不使用中间件，直接获取 body，在控制台会显示 undefined

  • 安装并引入 body-parser

    bodyParser.urlencoded 模块用于解析 req.body 的数据，解析成功后覆盖原来的 req.body，如果解析失败则为 {}。该模块有一个属性 extended，官方介绍如下：

    The extended option allows to choose between parsing the URL-encoded data with the querystring library (when false) or the qs library (when true). Defaults to true, but using the default has been deprecated.

    extended 选项允许配置使用 querystring(取值为false)或 qs(取值为true)来解析数据，默认值是 true，但这已经是不被赞成的了。
